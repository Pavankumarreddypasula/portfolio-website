Chatbot:

This project is a chatbot application built using Streamlit and Google Gemini (a generative AI model from Google). The purpose of the app is to allow users to input a question, send it to the Google Gemini model, and receive a response, which is then displayed in the app. The application maintains the conversation history so that users can view both their input and the AI's responses. The code starts by loading environment variables using load_dotenv(), which stores the Google Gemini API key securely in a .env file. This API key is then used to configure the Google Gemini API using the google.generativeai module.

The user interface (UI) is created using Streamlit, where the app header is set to "Gemini LLM Application". Users are provided with a text input field where they can type their questions, and once the question is submitted, the application sends it to the Gemini model. The model, specifically the gemini-1.5-flash version, generates a response, which is then displayed in the app. The chat history is maintained in the session state, allowing users to see the entire conversation history (both their own inputs and the AI's outputs).

The main function get_gemini_response() interacts with the Google Gemini model to generate responses. When a user submits a question, the model processes the input and returns a response, which is streamed back in real-time. The chatbot supports multiple interactions in one session, preserving the history of the conversation. The application layout is flexible, with the chat history being displayed below the input field, and each interaction is dynamically updated.

This chatbot could be customized for various domains, such as providing advice on applicant tracking systems (ATS), resume writing, or other professional topics. Additionally, it could be used for general Q&A applications, where users ask questions on a wide range of topics. To run this app, you need to have Python installed, along with the necessary libraries such as Streamlit, google-generativeai, and python-dotenv. You will also need to store your Google API key in a .env file for secure access. Once the app is set up, it can be run locally using the streamlit run command.

Overall, this chatbot application serves as a foundation for building more specialized AI-driven applications that can answer user queries dynamically and maintain a persistent conversation history.
